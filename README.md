# Task 2 – Java Developer Internship

## Project Title
Console-Based Java Calculator

## Description
This project is created as part of Task 2 of the Java Developer Internship.
It demonstrates Java primitive data types, variables, user input using Scanner,
basic arithmetic operations, type casting, and formatted output.

## Features
- Uses all Java primitive data types
- Accepts user input using Scanner
- Performs addition, subtraction, multiplication, and division
- Demonstrates type casting
- Handles invalid input like division by zero
- Shows local, instance, and static variables
- Well-commented Java code

## Tools Used
- Java
- IntelliJ IDEA (Mac M1)
- GitHub

## How to Run
1. Open Task2Calculator.java in IntelliJ IDEA
2. Click Run ▶️
3. Enter values in the console
4. View output
